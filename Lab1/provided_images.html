Импорт зависимостей и определение размеров доски¶

В качестве размеров указывается количество внутренних углов в шахматной сетке, которая используется для калибровки.

В данном случае в сетке 10 углов по вертикали и 14 по горизонтали.

Для каждого отдельного изображения в директории пытаемся найти углы шахматной доски.

Если желаемое число углов обнаружено, то уточняем координаты пикселей и отображаем их поверх шахматной доски.

В images храним пути файлов изображений. В lastImageWithCorners содержится последнее изображение шахматной доски с изображенными на ней углами доски.

In [42]:
images = glob.glob(imagesPath)
lastImageWithCorners = None
for fname in images:
    img = cv2.imread(fname, 1)
    gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

    ret, corners = cv2.findChessboardCorners(gray, CHECKERBOARD, cv2.CALIB_CB_ADAPTIVE_THRESH + 
                                             cv2.CALIB_CB_FAST_CHECK + cv2.CALIB_CB_NORMALIZE_IMAGE)
    
    if ret == True:
        objpoints.append(objp)
        
        corners2 = cv2.cornerSubPix(gray, corners, (11,11),(-1,-1), criteria)
        
        imgpoints.append(corners2)

        lastImageWithCorners = cv2.drawChessboardCorners(img, CHECKERBOARD, corners2, ret)
